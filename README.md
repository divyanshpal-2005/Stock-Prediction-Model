# Stock-Prediction-Model
This project is a simple full-stack web application that predicts stock prices using machine learning and visualizes market trends through interactive charts. The system fetches real-time historical stock data from Yahoo Finance and applies a Linear Regression model to estimate the next day’s stock price.

# AI Trading Dashboard 📈

## Features
- Stock price prediction using Linear Regression
- Interactive graph visualization
- Supports US & Indian stocks

## Tech Stack
- Python (FastAPI)
- HTML, CSS, JavaScript
- Chart.js

## How to Run
1. Install dependencies
2. Run:
   python -m uvicorn app:app --reload
3. Open index.html
